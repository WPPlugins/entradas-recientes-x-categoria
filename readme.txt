== Description ==

Widget basico que muestra las entradas mas recientes de una categoria seleccionada.

Las opciones de configuracion son:

*   "titulo" puedes cambiarle el titulo al plugin
*   "numero de post" En este campo puedes introducir un numero entre 1 y 15 para mostrar la catidad de post
*   "categoria" Aqui seleccionas una categoria para mostrar las entradas recientes.

   

== Installation ==

Instalacion del widget.

e.g.

1. Subir `entradas_recientes_x_categoria_widget.zip` al directorio `/wp-content/plugins/`
1. Activar el plugin en el menu 'plugins' de wordpress
1. Configurar el widget en el sidebar en la pestaña de widgets
